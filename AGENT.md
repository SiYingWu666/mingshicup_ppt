# 智能餐盘明石杯 PPT 协作说明

本仓库用于协作迭代“明石杯智能应用类比赛讲解视频用PPT”。最新推荐稿采用“保留型整合”：整体结构统一，章节内部尽量保留对应负责人原PPT的展示页、Demo页、流程图、表格和技术路线。

最终交付文件：

- `output/智能餐盘_明石杯讲解视频_整合版_保留原展示内容.pptx`

- `output/智能餐盘_明石杯讲解视频_整合版.pptx`

- `source/智能餐盘_机械结构设计部分.pptx`
- `source/视觉与联网部分_优化版.pptx`

## 最新页面结构

`智能餐盘_明石杯讲解视频_整合版_保留原展示内容.pptx` 为 28 页：

1. 封面：智能餐盘系统
2. 目录：四个章节总览
3. 系统总览：机械结构 + 传感采集 + AI识别 + 联网上传
4. 章节页：机械结构设计
5-10. 机械结构设计原PPT 6页保留型整合
11. 章节页：视觉识别与联网上传
12-16. 视觉与联网原PPT 5页保留型整合
17. 章节页：温度与重量传感器
18-25. 电控PPT核心技术路线 8页保留型整合
26. 章节页：实物功能性展示
27. 实物功能展示：称重、测温、识别、联网展示
28. 总结：机械承载、视觉Demo、柔性传感、云端记录形成闭环

章节顺序不要随意调整：

1. 机械结构设计
2. 视觉识别与联网上传
3. 温度与重量传感器
4. 实物功能性展示

## 保留型整合规则

- 机械部分以 `智能餐盘_机械结构设计部分.pptx` 为主，保留原6页展示结构。
- 视觉部分以 `视觉与联网部分_优化版.pptx` 为主，保留原5页展示结构和Demo感。
- 电控部分以 `智能餐盘_电控技术路线与创新点_V1(1).pptx` 为主，压缩但保留问题定义、路线升级、柔性传感、低噪声AFE、RA6M5、四层PCB、算法层和创新总结。
- 不要把有图片、表格、流程图、数据、Demo展示的原页面改成纯文字总结页。
- 只统一外层标题栏、页码、章节色、字体和版式对齐。
- 明显重复页面可以删除，但每个模块的核心展示逻辑必须保留。

## 必须保留的关键词

机械结构：

- 多层结构
- 滑槽安装
- 精准称重
- 功能集成
- 餐格 -> 垫块 -> 应变式传感器 -> 支撑层
- 摄像头固定视角
- 机械结构不是外壳，而是传感器和AI识别的稳定载体

视觉识别与联网：

- 让餐盘看懂食物，让饮食数据自动记录
- 摄像头采集 / 端侧识别 / 联网展示
- M85 / NPU / CEU / LCD
- YOLOv8n -> YOLO-Fastest -> 参数修改 / 数据集增强 / 端侧部署方案
- PC端验证效果 / LCD实机显示 / 端侧调试画面
- 30+ 常见菜品数据覆盖
- 主食类 / 素菜类 / 荤菜类 / 蛋豆类
- 饮食数据看板原型表格

温度与重量传感器 / 电控：

- 自制柔性多模态传感器
- PI柔性基底
- Cu/CuNi薄膜热电偶
- Pt薄膜应变单元
- 低噪声AFE
- RA6M5边缘采集
- 四层PCB
- 偏心误差补偿
- 温区判断
- 滤波、标定、进食状态机
